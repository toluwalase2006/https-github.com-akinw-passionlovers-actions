# PassionLovers

This is the PassionLovers dating platform built with React and TypeScript. The application allows users to create profiles, find matches, and interact with other users.

## Features

- User registration and authentication
- Profile creation and editing
- Matchmaking based on user preferences
- Messaging system for users to communicate
- Responsive design for mobile and desktop

## Project Structure

```
passionlovers
├── src
│   ├── components        # Contains reusable React components
│   ├── pages             # Contains main page components
│   ├── services          # Contains API service functions
│   ├── utils             # Contains utility functions
│   └── types             # Contains TypeScript interfaces and types
├── public
│   └── index.html        # Main HTML file
├── package.json          # npm configuration file
├── tsconfig.json         # TypeScript configuration file
└── README.md             # Project documentation
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd passionlovers
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the development server, run:
```
npm start
```
